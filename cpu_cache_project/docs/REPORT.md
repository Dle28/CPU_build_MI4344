# Dàn ý báo cáo

## 1. Giới thiệu

- Mục tiêu dự án
- Phạm vi học thuật
- Tóm tắt kiến trúc locked

## 2. Thiết kế ISA

- Định dạng lệnh
- Bảng opcode/funct
- Regfile và luật R0
- Quy tắc địa chỉ branch/jump

## 3. Datapath

- Pipeline 5 tầng (mục tiêu)
- Pipeline registers
- ALU và register file
- Luồng tín hiệu điều khiển

## 4. Xử lý hazard

- Forwarding paths
- Load-use stall
- Branch/jump flush
- Global stall khi cache miss

## 5. Hệ thống bộ nhớ

- Von Neumann unified memory path
- Quy tắc ưu tiên của arbiter
- Direct-mapped cache
- Main memory có delay

## 6. Kiểm thử

- Testbench từng module
- Test tích hợp
- Chương trình demo assembly
- Minh chứng waveform

## 7. Kết quả

- Tóm tắt tính đúng
- Ví dụ hit/miss cache
- Ví dụ stall/flush
- Giới hạn hiện tại

## 8. Kết luận

- Những gì đã triển khai
- Bài học rút ra
- Hướng phát triển ngoài phạm vi locked
